# Resources Folder

This folder should contain the TheAnimeScripter files for video processing.

## Setup

1. Download the latest release from [TheAnimeScripter](https://github.com/NevermindNilas/TheAnimeScripter/releases)
2. Extract the ZIP contents here
3. You should have files like `main.py`, `src/`, `Lib/`, `ffmpeg/`, etc.

After setup, the folder structure should look like:
```
resources/
├── main.py
├── src/
├── Lib/
├── ffmpeg/
├── Scripts/
└── ...
```
